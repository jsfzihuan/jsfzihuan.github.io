# 如何汇报自己的进展

汇报自己的进展是研究生生涯中重要的训练之一。如何向相关的老师汇报自己的进展呢？

####很简单，把自己当作老师，看看这样的汇报好不好？不要应付。

那应该怎么汇报呢？

以我的经历来讲最好是这样的步骤。

###最差：读了一篇文章，做一个PPT／word文档
应该包括一下内容：
1. 文章总结（自己的话）；
2. 文章用了什么方法；得到了什么结果；
3. 对自己的影响；
4. 文章存在什么不足

###应该：做一个PPT／word文档
应该包括一下内容：
1. 我目前做的课题大致题目是什么；
2. 我的数据是什么；
3. 我做了的结果是什么样，用了什么办法，得到了什么样的结果；
4. 存在什么问题

###最好：提交一份完整的论文。
完整的意思是：完全排好版的文章，就是可以投稿的文章，而不是少摘要，少参考文献，图片模糊等情况的发生。
